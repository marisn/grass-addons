DESCRIPTION

All inputs can be split in following categories (shown in separate tabs in GUI):

  • Inputs: input rasters with data and expected outcomes
  • SVM: static SVM parameters that do not change during the optimization process
  • Variables: parameters that can be optimized
  • Optimization: parameters governing the optimization process itself
  • Outputs: how to report optimization progress and outcomes
